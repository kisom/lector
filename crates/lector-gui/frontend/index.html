Keyboard Shortcuts

C-n / C-pScroll down / up (viewer) or next / prev (tree)
C-v / M-vPage down / up
C-f / C-bExpand / collapse (tree)
M-< / M->Beginning / end of document
TabToggle focus between tree and viewer
EnterOpen file / toggle directory (tree)
C-wClose current file
C-x C-fChange working directory
C-tToggle tree pane
C-= / C--Increase / decrease font size
C-0Reset font size
M-tCycle theme (Nord / eink / Tufte)
C-hToggle this help
q / C-x C-cQuit
EscapeDismiss dialog / cancel

Press Escape or C-h to close
